# What is iDM

iDM is a Download manager tool written in pyhton( Internet Download manager)

## Installation

git clone 

```bash
git clone https://github.com/abinvs-2019/iDM.git
```

## Usage
You will need a tool called ffmpeg to concatenate segments.
```python
pip install ffmpeg

```
